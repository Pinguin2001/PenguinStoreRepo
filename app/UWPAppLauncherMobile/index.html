Description

The whole point of this project is to provide the best traditional home screen experience as possible for Windows 10 devices.
Some people don't like Live Tiles so this is an alternative for them.
Also, apps load faster when opened with this launcher.
Note: This app is not on PenguinStore with the express permission from the developer and the download button just downloads the latest release from GitHub.
