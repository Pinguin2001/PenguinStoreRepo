Description

Fork of Unigram, which is an Telegram Client for Windows 10.

What´s new

v21.12.6006 - Pinned Messages 2.0: Pin multiple messages in any chat and see them all on a separate page
- Comment on posts in channels that have a discussion group.
- Get notified about replies to your comments via the new "Replies" chat (if you are not a member in the discussion group).
- Anonymous group admins: Turn on "Remain Anonymous" in an admin's permissions (edit group -> admins -> tap on an admin) to let them post on behalf of the group and become invisible in the list of members.
- Added a context menu button to the chat (in touch mode) for easier access to some context menus
- Added in-app volume control.
- Increased music player slider.
- Send music playlists
- Improved live locations
