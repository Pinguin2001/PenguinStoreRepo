Description

-It's fast and very handy to copy/paste from/to the terminal
-Tested on two SSH connections (Ubuntu server and SSH-W10M using Interop Tools)
-After each successful connection the data (IP,Port,User,Pass) will be saved automatically.
-IP/Host Address field support history and autocomplete.
Astifan added many improvements and enhancements to make it more usable especially for mobile made
GitHub link for the original project included in the app.
