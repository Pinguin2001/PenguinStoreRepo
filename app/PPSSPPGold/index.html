Description

PSP Emulator, compiled for Windows 10 Mobile by Astifan.
How to add folder to games list:
Manually by coping the files to (PPSSPP Package folder->LocalState->PSP->Games)
Why it's important:
You will be able to run custom roms (with EBOOT.PBP)
like 'RetroArch on PSP'
--------------------
IMPORTANT: If PPSSPP installed on your internal storage this could cause high disk usage
try not copy very large files
