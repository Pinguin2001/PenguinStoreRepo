Description

NotepadX is a simple texteditor.
With NotepadX you're be able to create standard text files, open text files and edit them.
The finished file can be sent directly from the app via E-Mail, stored locally or uploaded online to OneDrive.
Of course optimized for continuum, for free and without advertising.

Requirements

- Operating System: Win10 (10.0.10586.0)
- Architecture: ARM
